# 客户数据平台

## 简介

客户数据平台 （Customer Data Platform） 在GrowingIO 中被定义为一个数据中心 (Data Center) 。‌ 所有数据采集 、数据源接入，都会先汇集到客户数据平台，进行统一的数据融合与存储 。

为企业提供一个 **_统一的、持久的、可共享_** 的客户数据库，解决全渠道、多触点、多业务系统、线上线下的复杂业务中所面临的数据孤岛问题 。
