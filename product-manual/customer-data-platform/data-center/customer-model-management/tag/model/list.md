# 列表类的事件属性

## 简介

列表类的事件属性可以对用户产生事件的全部事件属性列表作为标签值，对用户打标。

常见使用场景为：

指定周期内，某个事件的

* 全部事件属性列表

控件说明：

![&#x5217;&#x8868;&#x7C7B;&#x7684;&#x4E8B;&#x4EF6;&#x5C5E;&#x6027;](../../../../../../.gitbook/assets/image%20%28200%29.png)

| 项 | 说明 |
| :--- | :--- |
| 1.选择时间 | 如过去7天、过去30天、过去90天等 |
| 2.选择事件 | 如全局指标\(访问、活跃\)和埋点事件 |
| 3.选择属性 | 如埋点事件的字符串类型事件属性 |
| 4.选择事件过滤 | 选择事件过滤条件 |
