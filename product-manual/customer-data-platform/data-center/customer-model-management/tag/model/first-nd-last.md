# 首次/末次的事件属性

## 简介

首次/末次的事件属性可以对事件第一次或最后一次发生的字符串类型事件属性作为标签值，对用户打标。

常见使用场景为：

指定周期内，某个事件的

* 首次/末次 发生时的事件属性
* 首次/末次 发生时距今天数
* 首次/末次 发生时的具体日期

## 控件说明：

![](../../../../../../.gitbook/assets/image%20%28556%29.png)

| 项 | 说明 |
| :--- | :--- |
| 1.选择时间 | 如过去7天、过去30天、过去90天等 |
| 2.选择计算模型 | 如最初、最终 |
| 3.选择事件 | 如全局指标\(访问、活跃\)和埋点事件 |
| 4.选择属性 | 如距今天数、日期和埋点事件的字符串类型事件属性 |
| 5.选择事件过滤 | 选择事件过滤条件 |
