# 项目列表

## 接口说明
查询项目列表

## 接口地址
http://{api-host}/v1/api/projects

## 请求方式
GET


## 公共请求参数
[公共请求参数](../../open-api#公共请求参数)

## 请求参数
| 名称 | 类型 | 必填 | 描述 | 示例值 |
| --- | --- | --- | --- | --- |
| 无 |  |  |  |  |

## 返回数据

| 名称 | 类型 | 描述 |
| --- | --- | --- |
| id | string | 项目ID |
| name | string | 项目名称 |
